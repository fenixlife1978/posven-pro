# POSven Pro

Scaffold inicial para POS de licorería.

Este repositorio contendrá la aplicación POSven Pro: frontend (Next.js + Tailwind) y backend (Node.js + Prisma + PostgreSQL).

Estructura prevista:
- apps/backend
- apps/frontend
- docker-compose.yml

Instrucciones rápidas:
1. El proyecto se subirá en la rama `feat/posven-initial` por la asistente.
2. Revisa los commits en la rama y abre PR cuando estés listo para revisar.

Notas:
- Precio base en USD. Equivalente en BS calculado con tasa BCV manual.
- Si deseas que cree la rama feature y suba el scaffold, la asistente procederá ahora.
